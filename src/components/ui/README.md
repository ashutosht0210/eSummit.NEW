# UI Components (`src/components/ui/`)

This directory contains atomic, reusable presentational UI elements designed specifically for the E-Summit 2026 application.

## Files Added

- **`Button.jsx`**: Core E-Summit themed button supporting variants (`primary`, `secondary`, `outline`, `signal`) and size options.
- **`TransitionLink.jsx`**: Wrap standard `<a>` links to apply dynamic GSAP transitions before routing.
- **`Input.jsx`**: Form text input field matching the dark, monospaced theme.
- **`Select.jsx`**: Stylized dropdown selector field.
- **`Field.jsx`**: Labeled parent container for input fields.
- **`Modal.jsx`**: Core overlay modal with escape-key handling, fade-in animations, and scroll lock.
- **`PageHeader.jsx`**: Top-of-page header layout with monospaced tags.
- **`SectionHeader.jsx`**: Section header supporting vertical and split-screen grid layouts.
